# 🪷 Jai Shri Ram — Japa Counter

A devotional Android app for counting Ram Naam Japa with daily habit tracking.

## Features
- 🪷 **Charan Sparsh** — tap Ram's lotus feet to count
- 📿 **Rudraksh Mala** — spins with every jap, beads light up
- 🤫 **मन जप** — silent mental chanting mode (touch only)
- 🎙️ **वाणी जप** — voice chanting mode, on-device speech recognition
- 🔥 **Daily streak** tracker
- 📊 **90-day history**
- 🎯 **Daily goal** (default 108, customisable)
- 🎵 Background meditative music

## Voice Recognition
Works offline using Android's on-device speech engine.
Recognises all dialects and variations of Ram's name:
Ram, Raam, Rama, Shri Ram, Jai Ram, Jai Shri Ram, Siyaram, and more.
Works in noisy environments, whispers, and loud chanting.

## Assets needed (add to assets/ before first build)
- `assets/images/ram_charan.png` — HD image of Ram's lotus feet
- `assets/images/temple_bg.png` — temple background image
- `assets/images/icon.png` — app icon (1024x1024)
- `assets/images/splash.png` — splash screen (1284x2778)
- `assets/images/adaptive-icon.png` — Android adaptive icon (1024x1024)
- `assets/audio/ram_chant.mp3` — Ram chant sound on each jap
- `assets/audio/bg_music.mp3` — looping meditative background music

## Build
Every push to `main` triggers an automatic APK build via GitHub Actions.
Download the APK from the Actions tab → latest run → Artifacts.
